To create basic Docker app:
1. Create app folder: app.js and views/home.ejs (port = 3000)
2. npm init -y
3. npm i express ejs
4. Dockerfile outside app folder:
FROM node:14-alpine

COPY /app /app

WORKDIR /app

CMD [ "node", "app.js" ]

5. docker build -t practice-app . (run in Dockerfile directory)
6. docker run -p 3000:3000 practice-app
7. docker stop [container name checked with docker ps -a] and docker start [container name] to restart the application

If app doesn't update with code change, remove container and image and rebuild
